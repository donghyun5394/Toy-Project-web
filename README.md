# 딩고-코딩문고
# 주요 기능

## 메인

- 도서 검색
    - 도서 정보 조회
- 이벤트 배너
    - 도서 구매 시 사은품 증정 배너
- 베스트셀러
    - 평점과 판매량 순위 1,2위 표시
- 검색창
    - 도서 검색기능 구현

## 전체 도서 목록

- (가나다)순 정렬
    - 도서 제목순 정보 조회
- 날짜순 정렬
    - 출간 날짜순  정보 조회

## 추천 도서 목록

- 평점 순위 추천
    - 평점 수치에 따른 도서 조회
    - 평점 순 도서 목록 정렬
- 판매량 순위 추천
    - 최다 판매량 도서 조회
    - 판매량 순 도서 목록 정렬

## 도서 정보

- 도서 기본 정보
    - 도서 소개
- 리뷰 창
    - 이전 작성 리뷰 확인
- 리뷰 추가
